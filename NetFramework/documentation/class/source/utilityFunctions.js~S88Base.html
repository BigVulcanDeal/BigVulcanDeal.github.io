    S88Base

    Direct Subclass:

    AnyType, ApprovalHistoryType, BatchInformationType, BatchListEntryType, BatchListType, BatchSizeType, ClassInstanceAssociationType, ConstraintType, ControlRecipeType, EnumerationSetType, EnumerationType, EquipmentConnectionType, EquipmentElementType, EquipmentIDType, EquipmentProceduralElementClassType, EquipmentProceduralElementType, EquipmentRequirementType, FormulaType, FromIDType, HeaderType, IndividualApprovalType, LinkType, ListHeaderType, MasterRecipeType, ModificationLogType, OtherInformationType, ParameterType, ProcedureLogicType, PropertyType, RecipeBuildingBlockType, RecipeElementType, StepType, ToIDType, TransitionType, ValueType

    The S88Base class is the base class for all of the core objects that are used to define an S88 model. It mainly supports generic logic for

    • reading and writing the object tree in BatchML format
    • reading and writing the object tree in JSON format
    • performing intelligent copies of one object to another (e.g. generating a ControlRecipe from a MasterRecipe)
